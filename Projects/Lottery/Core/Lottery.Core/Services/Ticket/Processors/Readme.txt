1. 2D Đầu
- Miền Bắc: 2D Đầu đánh các số giải 7 (gồm 4 số) => Số tiền x 4.
- Miền Trung + Name: 2D Đầu đánh các số giải 8 (1 số) => Số tiền x 1.

2. 2D Đuôi
- Miền Bắc + Trung + Name: Cược 2Đ Đuôi của giải Đặc biệt => Số tiền x 1.

3. 2D Đầu Đuôi
- Miền Bắc: Đánh các số của giải 7 (4 số) & giải đặc biệt (1 số) => Số tiền x 5.
- Miền Trung + Name: Đánh các số của giải 8 (1 số) & giải đặc biệt (1 số) => Số tiền x 2.

4. 2D27 Lô / 2D18 Lô
- Tên gọi khác: 2D Bao lô
- Miền Bắc (2D27 Lô): Đánh các số của các giải (27 số) => Số tiền x 27.
- Miền Trung + Nam (2D18 Lô): Đánh các số của các giải (18 số) => Số tiền x 18.

5. 3D Đầu
- Miền Bắc: Đánh các số giải 6 (gồm 3 số) => Số tiền x 3.
- Miền Trung + Nam: Đánh các số giải 7 (gồm 1 số) => Số tiền x 1.

6. 3D Đuôi
- 3D Đuôi đánh số giải đặc biệt => Số tiền x 1.

7. 3D Đầu Đuôi
- Miền Bắc: 3D Đầu Đuôi đánh các số giải 6 & giải đặc biệt => Số tiền x 4.
- Miền Trung + Nam: 3D Đầu Đuôi đánh các số giải 7 & giải đặc biệt => Số tiền x 2.

7. 3D23 Lô / 3D17 Lô
- Miền Bắc: Đánh 23 số (các số có ít nhất 3 chữ số) => Số tiền x 23.
- Miền Trung + Nam: Đánh 17 số (các số có ít nhất 3 chữ số) => Số tiền x 17.

8. 4D Đuôi
- Miền Bắc + Trung + Nam: Đánh số giải đặc biệt => Số tiền x 1.

9. 4D20 Lô / 4D16 Lô
- Tên gọi khác: 4D Bao lô
- Miền Bắc: Gồm 20 số (các số có ít nhất 4 chữ số) => Số tiền x 20.
- Miền Trung + Nam: Gồm 16 số (các số có ít nhất 4 chữ số) => Số tiền x 16.

Note:
